# Tech Store - React E-commerce

A modern, responsive, and feature-rich e-commerce application built with **React**, **Vite**, and **Material UI**. This project showcases a clean architecture and a premium user experience for an online shopping platform.

## 🚀 Features

- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Dark/Light Mode**: Seamless theme switching using Material UI and custom context.
- **Modern UI Components**: Built with Material UI for a sleek and professional look.
- **Interactive Slides**: Integrated Swiper for beautiful hero sections and product carousels.
- **Scroll to Top**: Smooth navigation experience.
- **Structured Codebase**: Organized component folder structure for easy maintenance and scalability.

## 🛠️ Tech Stack

- **Frontend**: [React](https://react.dev/) (v19)
- **Styling**: [Material UI (MUI)](https://mui.com/), [Emotion](https://emotion.sh/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Sliders**: [Swiper](https://swiperjs.com/)
- **State Management**: React Hooks & Context API

## 📁 Project Structure

```text
src/
├── components/
│   ├── footer/      # Footer component
│   ├── header/      # Header levels (Top, Mid, Navigation)
│   ├── hero/        # Hero section and sliders
│   ├── main/        # Product display and categories
│   └── scroll/      # Scroll to top functionality
├── images/          # Static assets
├── theme.jsx        # MUI theme configuration and dark/light mode logic
├── App.jsx          # Root component
└── main.jsx         # Entry point
```

## ⚙️ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <https://github.com/Latreche-khalil14/React-E-commerce>
   ```
2. Navigate to the project directory:
   ```bash
   cd React-ecomerce
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the production bundle:
```bash
npm run build
```
